# 真實接入：daemon agent runtime — 設計與實測

- 日期：2026-05-25
- 構件：`connectors/agora-runtime/`（echo 的真正繼任者）
- 一句話：把「只會 echo / 只暴露工具」升級成「會真的處理脈絡」的常駐 agent 執行期。

## 問題

上線在接的都是 echo，沒有真正去處理內容。真正要的是：當 OpenClaw/Hermes 進房，被 @ 時才把「mention + 相關(relative)」的脈絡納入它的 session，當作它"visible"的內容 → 降噪 + 有跡可循。

## 設計（已落地）

迴圈：**喚醒 → 組裝可見脈絡（有界圖閉包）→ backend 推理 → 帶出處回寫 → 游標推進（不斷層、可重放）**

關鍵決策：

1. **可見脈絡 = 有界圖閉包**。觸發訊息沿 `reply_to`(上文鏈)/`replies`(直接回覆)/`relates`(引用) 展開，套 max 則數 + token 預算 + 摘要降級 + 去重（`src/assembler.js`）。hub 留全量，agent 看到的是可再展開的投影。
2. **WSS 只當喚醒訊號，資料一律走 `inbox(since=cursor)` 有序補讀** → 有序、無斷層、可重放。
3. **以 hub `cursor_seq` 為權威，處理成功才推進** → 至少一次 + 重啟不重複。
4. **穩定身分**：首次 `/pair` 後把 `{agentId, token}` 落地 state 檔，重啟用同一身分重連（不重新 pair），游標才接得上。
5. **出處回寫**：回話 `reply_to`=觸發訊息、`relates`=實際取用脈絡 → 圖連通、可稽核。
6. **可插拔 backend**：`mock`（測試全實作，不接 LLM 但真的讀脈絡並標出處）、`openclaw`/`hermes`（adapter 縫，套通用 `httpAgent`：MCP 信封拆包 + 送出去重 + session key 慣例 `agora:<廠牌>:<room>:<agent>`）。唯一整合縫＝一個收 `{sessionKey,input}` 回 `{content}` 的 gateway turn 端點。

## 實測（沙盒，對 node:sqlite 版 hub）

情境：人類先發「背景訊息」（不 @room，僅作上文）→ runtime(mock) 入房 → 人類 `@runtime` 且 `reply_to` 背景訊息 → 殺掉 runtime → 停機時再發一則 → 同身分 resume。

| 驗證項 | 結果 |
|---|---|
| 被 @ 後喚醒並處理 | PASS |
| 組裝撈到 reply_to 的上文（背景訊息） | PASS |
| 回覆 `reply_to` = 觸發訊息 | PASS |
| 回覆 `relates` 含被引用的背景訊息（出處） | PASS |
| 人類觸發 → 回覆定址 `@room` | PASS |
| 重啟以**同一 agentId** 重連（憑證持久） | PASS |
| 重啟後游標接 `cursor_seq=2`，補齊停機期間的提問（不斷層） | PASS |
| 不重複處理舊訊息（恰好一次） | PASS |

最終 transcript（節錄）：

```
#2 翰[human]  to=[runtimeId]  reply_to=#1 : @OpenClaw-RT 依上面的背景，外牆模板扣除原則…

```

## 還沒做 / 限制

- **真實 OpenClaw/Hermes 未端對端串接**：它們在翰的主機、我的沙盒連不到，故做成有合約、可插拔的 adapter 縫；翰提供 gateway turn 端點（設 `OPENCLAW_GATEWAY_URL`/`HERMES_URL`）即可接上。沙盒只用 mock backend 驗證迴圈。
- 摘要降級目前是「省略並列出 ids」的結構化提示，非語意摘要（要語意摘要可由 backend 端做）。
- 斷鏈引用（沒設 reply_to/relates 的隱含脈絡）靠 backend 自行判斷缺脈絡時明說 + 可選 view_session 逃生口；尚未加自動語意檢索增補。

## 怎麼接真的 OpenClaw（給翰）

1. 讓 OpenClaw gateway 暴露一個 turn 端點，收 `{ sessionKey, input }` → 灌進對應 session 跑一回合 → 回 `{ content }`。
2. gateway 設 `tools.sessions.visibility:"all"` + `gateway.tools.allow`。
3. `export OPENCLAW_GATEWAY_URL=...`，用 `--backend openclaw --kind openclaw` 啟動 runtime，拿 GUI 邀請碼首次加入即可。
